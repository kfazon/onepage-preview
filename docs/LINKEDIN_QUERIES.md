# LinkedIn Sales Navigator Search Configurations

Step-by-step instructions for running outreach prospect searches on LinkedIn Sales Navigator for the DE (Germany) market.

---

## Overview

LinkedIn Sales Navigator is the primary tool for finding **German small business owners and decision-makers** who are active on LinkedIn. Use these exact filter configurations per vertical.

**Access:** `https://www.linkedin.com/sales/search/people`

---

## Shared Filter Settings (All DE Verticals)

| Filter | Value |
|---|---|
| **Location** | Germany (or specific cities listed per vertical) |
| **Company Size** | `1-10`, `2-10` (small/local businesses) |
| **Industry** | See per-vertical settings below |
| **Title Keywords** | `Inhaber`, `Geschäftsführer`, `Besitzer`, `Inhaberin`, `Geschäftsführerin` |

### Filter Configuration

| Filter | Value |
|---|---|
| Location | Germany: Berlin, Hamburg, Munich, Köln, Düsseldorf, Frankfurt, Stuttgart, Leipzig |
| Company Size | `1-10` |
| Industry | `Consumer Services` |
| Title Keywords | `Inhaber`, `Geschäftsführer`, `Besitzer`, `Inhaberin`, `Salon`, `Friseur` |
| Company Name Keywords | `Friseur`, `Hair`, `Salon`, `Kosmetik`, `Schönheit`, `Beauty`, `Pfleger` |

### Specific DE Cities
- **Berlin:** Bezirk Mitte, Prenzlauer Berg, Kreuzberg, Neukölln
- **Hamburg:** Eimsbüttel, Altona, Winterhude, Harvestehude
- **Munich:** Maxvorstadt, Schwabing, Haidhausen, Giesing
- **Köln:** Altstadt-Süd, Neustadt-Nord, Lindenthal, Ehrenfeld
- **Düsseldorf:** Altstadt, Flingern, Unterbilk, Pempelfort
- **Frankfurt:** Innenstadt, Sachsenhausen, Nordend, Westend

### How to Find Decision-Maker Profiles
1. Search by title keyword `Inhaber` + company name keyword `Friseur`
2. Look for profile headline: `Inhaberin — Friseur Studio X`
3. Check `All` connections → filter to `Second Degree` to find warm intros
4. Look for profiles with 1–3 previous jobs (linear career = likely sole owner)

### Outreach Message Template (DE)
```

Hättest du 5 Minuten für einen kurzen Chat?

LG
```

### Follow-up Template
```

```

## LinkedIn Outreach Best Practices

### Profile Optimization Before Outreach
- Use a **Premium** or **Sales Navigator** account (required for advanced filters)
- Ensure your own LinkedIn profile looks professional and mentions helping local businesses
- Use `Sales Navigator` browser extension for inmail tracking

### Message Timing
- **Best days:** Tuesday–Thursday
- **Best times:** 8–9am or 5–6pm (when targets are checking LinkedIn)
- **Follow-up:** Send 1 follow-up message 3–4 days later, then stop

### Connection Request Strategy
1. Send **connection request** with personalized note (max 300 chars)
2. After connecting, send **InMail** or LinkedIn message

**Connection Request Note Template (DE):**
```
Hi [Vorname], ich unterstütze kleine lokale Geschäfte dabei, ohne eigene Website online sichtbar zu sein — vielleicht relevant für dich?
```

### Segmentation by City (Suggested Priority)
| City | Why |
|---|---|
| Berlin | Largest market, many independent businesses |
| Hamburg | Affluent, high SMB density |
| München | High spending power, many owner-operated shops |
| Köln | Third largest, strong local scene |
| Düsseldorf | Premium retail and dining |
| Frankfurt | Finance-adjacent businesses |

### Anti-Spam / Safety
- Do NOT send mass outreach without personalization — LinkedIn will shadowban your account
- Keep daily outreach to **50–80 connection requests** maximum
- Vary message copy slightly per batch to avoid spam detection
- Do NOT use the word `kostenlos` (free) too many times — LinkedIn scans for promotional language
