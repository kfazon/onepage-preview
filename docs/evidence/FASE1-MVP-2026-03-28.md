# Evidence — Faza 1 MVP (2026-03-28)

## Što je isporučeno

### Kompletna landing page
- URL: `https://github.com/kfazon/onepage-preview`
- Build: prolazi ✅ (`pnpm build` — Astro 5 + Cloudflare Pages adapter)
- Stranica: `/` — full marketing landing page s:
  - Hero section s CTA
  - 3 template preview showcase (iframe embed)
  - Pricing tiers (Preview / Go Live / Growth)
  - Ethical outreach banner (opt-out, public data disclosure)

### Generator MVP
- `packages/generator/src/index.js` — 3 template enginea:
  - `launch-teaser` — za nove biznise, konzultante
  - `product-spotlight` — za single-offer biznise
  - `event-waitlist` — za evente, worshopy, pop-up
- Claim/opt-out footer na svakoj generiranoj stranici
- `POST /api/preview` — programmatic HTML generation
- `GET /api/preview?name=...&template=...` — live preview from query params

### API Endpoints
- `POST /api/claim` — email capture za upgrade interest
- `POST /api/optout` — brisanje iz outreach pipelinea
- JSON store (`data/previews.json`) za MVP (swap za Cloudflare KV u produkciji)

### Demo page
- `/demos` — live iframe showcase sva 3 templatea + API poziv primjeri

## Build artifact
