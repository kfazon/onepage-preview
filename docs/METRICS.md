# Metrics

## Purpose
Define what success/failure looks like and how it is measured.

## Baseline operational metrics
- Delivery lead time
- Change failure rate
- MTTR (mean time to recovery)
- Reliability of critical paths

## Reporting discipline
- Update key metrics at least once per milestone.
- Note major metric deltas and likely causes.

## Product KPIs

- preview generation success rate
- p95 generation latency
- cache/KV hit ratio
- locale-specific render errors
- publish-to-live lead time
