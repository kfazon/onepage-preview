# onepage-preview

Monorepo for one-page preview generation (Peek product).

## Quickstart

```bash
pnpm install
pnpm --filter @onepage/web dev
```

## Config
- Cloudflare Pages deployment
- KV namespace for previews
- Stripe for payments
- Resend for email
